# Run Notes

Use this directory for run-specific notes that should not be promoted to durable repo memory yet.

Suggested naming:

- `YYYY-MM-DD-short-topic.md`

Examples:

- `2026-04-06-intent-service-failures.md`
- `2026-04-06-cli-orchestrator-integration.md`

Promote stable findings from run notes into:

- `wiki/memory/repo-facts.md`
- `wiki/open-issues.md`
- subsystem pages under `wiki/architecture/`
